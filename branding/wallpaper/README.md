# SpaghettiOS Wallpapers

Place your wallpaper images here. Recommended formats:

- **Desktop wallpaper**: 3840x2160 PNG or JPEG
- **GRUB background**: 1920x1080 PNG
- **Plymouth background**: 1920x1080 PNG
- **Login screen**: 1920x1080 PNG

## Default wallpapers to create:

- `spaghetti-desktop.png` — Desktop background
- `grub-background.png` — GRUB menu background
- `plymouth-wallpaper.png` — Boot splash background
- `login-background.png` — Login screen (GDM/lightdm)

Design suggestion: Rich italian red (#8B0000) gradient with golden (#FFD700)
spaghetti strands forming patterns, with the SpaghettiOS logo centered.
